# Sobre a plataforma
[Beecrowd](https://www.beecrowd.com.br) é uma plataforma que conta com diversos exercícios, divididos em 9 grandes categorias, para ajudar estudantes com exercícios de lógica aplicadas em diferentes tópicos.
